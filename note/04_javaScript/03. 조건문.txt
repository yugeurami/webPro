[조건문]
기본적으로 if문이나 삼항연산자 switch문은 java와 사용하는 방식이 같다
다른 점은 짧은 조건문이 가능하다


<짧은 조건문>
논리 연산자의 특성을 조건문으로 사용한다
논리합 연산자를 사용한 표현식은 뒤에 어떠한 값이 들어가도 항상 참임으로 우변을 실행하지 않는다

true || 항상 실행할 문장
false || 실행하지 않을 문장

<script>
        var input = Number(prompt('숫자를 입력하세요', '숫자'));
        input % 2 == 0 || alert('홀수입니다');
        input % 2 == 0 && alert('짝수입니다');
</script>
