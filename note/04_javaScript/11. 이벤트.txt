[이벤트]

<정의>
자바스크립트에서 DOM을 통해 동적인 화면 처리를 위해 처리하는 내용
키보드를 이용해 버튼을 입력하거나 마우스 클릭과 같이 다른 것에 영향을 미치는 것을 의미한다

​

<종류>
마우스 이벤트, 키보드 이벤트, HTML 프레임 이벤트, 입력 양식 이벤트, 유저 인터페이스 이벤트, 구조 변화 이벤트, 터치 이벤트 
ex. onclick, onmouseover, onmouseout, onkeyup(키보드의 특정키를 클릭했다가 손가락을 벗어났을 때)


1.인라인 이벤트 모델
해당 태그에 바로 이벤트를 걸어준다
​

2. 기본 이벤트 모델
on으로 시작하는 속성(이벤트 관련 속성)에 function을 할당해서 사용하는 것


3. 표준 이벤트 모델
이벤트와 이벤트 리스너 함수(function)를 연결
​


<이벤트 제거>
이벤트를 특정한 조건에 맞춰서 그만두게 만드는 것
일부 HTML 태그는 이미 이벤트 리스너가 있는데 이를 디폴트 이벤트라고 한다 
이런 디폴트 이벤트 역시 제거가 가능하다

​

<이벤트 제한>
조건을 걸어 특정 조건일 때만 이벤트를 발생하도록 한다
디폴트 이벤트에도 제한을 걸 수 있다

​

<이벤트 강제 발생>
하나의 이벤트를 다른 이벤트와 연결해서 강제로 발생하게 만든다


​

<이벤트 전달(이벤트 버블링 방식)>
어떤 이벤트가 먼저 발생하면 순차적으로 이벤트를 발생시키는 것을 이벤트 전달이라고 한다
자바스크립트에서는 자식 노트에서 부모 노트 순으로 이벤트를 실행한다

​

<이벤트 객체>
1. 문서 로딩 이벤트
onload	문서 로딩이 끝나면 이벤트 발생
onunload	문서에서 벗어날 때 이벤트 발생
onabort	문서가 완전히 로딩되기 전에 불러오는 것을 멈추면 이벤트 발생
onerror	문서가 정확하게 로딩되지 않으면 이벤트 발생

​

2.마우스 이벤트
onclick		마우스 클릭 시 이벤트 발생
ondbclick		마우스 더블 클릭 시 이벤트 발생
onmousedown	마우스 버튼을 눌렀을 때 이벤트 발생
onmouseup	마우스 버튼에서 손을 떼면 이벤트 발생
onmousemove	마우스 포인터를 움직일 때 이벤트 발생
onmouseover	마우스 포인터가 요소 위에 오면 이벤트 발생
onmouseout	마우스 포인터가 요소를 벗어나면 이벤트 발생
​

3. 키보드 이벤트
onkeydown	키를 누르는 동안 이벤트 발생
onkeypress	키를 누를 때 이벤트 발생
onkeyup		키에서 손을 떼면 이벤트 발생

​
4. 폼 이벤트
blur	폼 요소에 포커스를 둘 때 이벤트 발생
change	목록이나 체크 상태 등이 변경되면 이벤트 발생
	<input>,<select>,<textarea>태그에서 사용
focus	폼 요소에 포커스가 놓였을 때 이벤트 발생.
	<label>,<select>,<textarea>,<button>태그에서 사용
reset	폼이 리셋되었을 때 이벤트 발생
submit	submit 버튼 클릭시 이벤트 발생
