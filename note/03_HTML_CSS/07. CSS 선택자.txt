[선택자]
HTML 문서 중 특정 부분을 선택하여 해당 부분의 속성을 변경하기 위해 사용한다
여러 선택자를 동시에 사용할 수 있다


1. tag 선택자
특정 태그를 선택하여 CSS속성을 적용한다


2. 전체 선택자(*)
문서 전체를 선택한다


3. id(#)와class(.) 선택자
태그의 id와 class를 이용하여 CSS에 속성을 설정할 수 있다


4. 속성 선택자
tag의 속성을 선택하여 CSS의 속성을 설정할 수 있다

선택자[속성]	지정한 속성을 가지고 있는 요소를 찾아 스타일을 적용
선택자[속성=값]	속성 값이 특정값과 같은 경우를 찾아 스타일을 적용
선택자[속성^=값]	속성 값이 지정한 문자로 시작하는 속성값에 대해서만 스타일을 적용
선택자[속성$=값]	속성 값이 지정한 문자로 끝나는 속성에 대해서만 스타일을 적용


5. 후손 및 자손 선택자
(1)후손 선택자
태그를 선택할 때 한칸을 띄어 쓰면 그 태그 밑에 있는 선택자에 해당하는 모든 태그를 의미한다

(2)자손 선택자(>)
앞에 있는 태그 바로 아래에 있는 태그를 의미한다


6. 동위 선택자
동등한 위치에 있는 태그를 선택한다

(1) +
앞의 태그 바로 뒤에 오는 한 요소

(2) ~
앞의 태그 다음에 오는 형제 요소 전부


7. 반응 선택자(:hover)
마우스를 올려 놓았을 때의 반응에 따른 속성을 설정할 수 있다


8.상태 선택자
상태에 따라 CSS속성이 변화하는 설정을 할 수 있다


9. 구조 선택자
:nth-child(2n+1)	홀수번째
:nth-child(2n)	짝수번째
:first-child	첫번째
:last-child		마지막


10. 문자 선택자
특정 문자를 선택하여 CSS 속성을 설정

::first-line		첫문장
::first-letter	첫글자
::selection	문단


11. 링크선택자
문서에 링크(href) 되어 있는 문자를 선택하여 CSS 속성을 설정


12. 부정 선택자( :not() )
나를 제외한 모든 태그에 CSS 속성을 적용
