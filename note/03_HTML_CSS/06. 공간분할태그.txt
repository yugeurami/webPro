[﻿공간분할 태그(레이아웃태그)﻿]
웹페이지를 만들 때 기본적으로
header(회사 이름이나 로고등)
nav(메뉴)
content(페이지의 콘텐츠부분)
footer(회사정보 등)
으로 공간을 분할한다


1. div
block 형식으로 공간을 분할
하나의 블록형식의 태그므로 개행을 포함한다

그 외 블록형 태그
<p></p>
-----------------------------
<ul>
<li></li>
</ul>
-----------------------------
<table>
<tr>
<td></td>
</tr>
</table>


2. span
inline 형식으로 공간을 분할


3. 시멘틱 태그
스타일 시트를 입력할 때 태그의 기능을 바로 알아 볼 수 있도록 하는 태그
유지보수 시 효율적이다

header	해더
nav	내비게이션
aside	사이드에 위치하는 공간
section	여러 중심 내용을 감싸는 공간
article	글자가 많이 들어가는 부분
footer	꼬리말
