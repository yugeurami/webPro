[CSS3]
CSS를 HTML 문서에서 분리할 수 있다
CSS3에서 @로 시작하는 키워드를 규칙이라고 한다

1. @import
link를 대신할 수 있다
<style>태그 내부의 상단에 위치해야한다

<style>
	@import url(‘title.css’);
</style>


2. @media
media 속성을 이용해 사용 환경에 따른 CSS를 적용할 수 있다
디바이스의 종류는 상당히 많고 단말기의 사이즈도 다양하다
웹페이지를 만들면 디바이스에 따라 달리 보여야 한다

all		모든 미디어 타입
handheld		손으로 들고 다니면서 볼 수 있는 작은 스크린에 대응하는 용도
print		인쇄 용도
projection	프로젝터 표현 용도
screen		컴퓨터 스크린을 위한 용도
tv		음성과 영상이 동시 출력되는 TV와 같은 장치
