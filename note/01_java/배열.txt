[배열]
레퍼런스 변수 타입(값 대신 주소가 들어감)
동일 자료형의 집합
하나의 이름으로 여러 개의 데이터를 사용할 수 있음
여러 개의 값을 모아 놓은 하나의 변수

배열의 크기는 최초에 한번 설정되면 변경이 불가능

<배열 선언>
int [] 변수명 = new int [length];
배열 변수를 선언하고 값을 설정 안해주면 공간에 0이 들어가있음(레퍼런스 변수)
int [] 변수명 = {값들을 넣어줌};
배열의 index는 0부터 시작


<확장 for문>
배열 출력 시에 사용
for(int temp : arr) {
	System.out.println(temp);
}

<API를 이용한 배열 복사>
System.arraycopy(원본,원본시작idx,복사본배열,복사본시작idx,방갯수)
