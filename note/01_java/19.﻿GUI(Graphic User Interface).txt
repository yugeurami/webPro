[GUI]
아이콘처럼 사용자가 사용하기 편리하게 그래픽 요소를 가미하여 컴퓨터를 사용하기 쉽게 만들어 놓은 것
↔ CUI(Character User Interface) : cmd창

<﻿AWT>
JAVA에서 그래픽 요소를 만들기 위한 컴포넌트들을 의미한다.
ex.버튼, 체크박스 등

컴포넌트의 배치와 관련된 클래스
BorderLayout : 기본 Layout. 컴포넌트를 순서대로 하나씩 넣는다
GridLayout : 행열을 설정하고 컴포넌트를 1행 1열 부터 하나씩 넣는다, 모든 칸의 크기는 동일하다
FlowLayout : 왼쪽에서 오른쪽으로 배치되며 오른쪽에 더 이상 공간이 없으면 다음 줄로 자동 배치한다

그래픽 출력과 관련된 클래스
Color, Font, Rectable, Point, Graphics, Image

컨테이너 클래스 그외의 클래스
Frame,Panel, Window, Container, FileDialog, Demension, Event


<﻿Swing>
Swing은 AWT보다 나중에 만들어져 복잡하고 화려한 컴포넌트를 제공한다
