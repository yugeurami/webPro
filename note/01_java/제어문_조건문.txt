<조건문>
1. if문
조건을 설정하고 조건에 맞지 않으면 실행하지 않음

if(조건식1) {
	조건식1이 참일 때 실행할 명령문;
}else if(조건식2){
	조건식1이 거짓이고 조건식2가 참일 때 실행할 명령문;
}else{
	위의 조건식 둘 다 참이 아닐 때 실행할 명령문;
}

﻿if문 안에 if문을 넣을 수 있다!!

+ 블럭 안에서 선언한 변수는 블럭 내부에서만 작동함


2. switch문
if~else if의 친구
가독성 면에서 좋음
정수형/문자형/문자열 변수만 가능함

switch(변수){
	case 변수값1 : 위의 변수가 변수값1일 때 처리할 명령문; break;
	case 변수값2 : 위의 변수가 변수값2일 때 처리할 명령문; break;
	∙ ∙ ∙ ∙ ∙ ∙
	default : 위의 변수값들이 아닐 때 처리할 명령문; break;

break; 꼭 넣기!
